# لیست جامع دارایی‌های صوتی (Audio Assets) جهت ضبط یا TTS

این سند شامل تمامی فایل‌های صوتی مورد نیاز پروژه **چیستان‌آباد** است. برای بهترین خروجی، صداها باید با لحنی آرام، مهربان و تشویقی ضبط شوند.

**تنظیمات پیشنهادی برای خروجی TTS:**
- **زبان:** فارسی (ایران)
- **سرعت:** ۰.۹ (کمی آهسته‌تر از نرمال)
- **فرمت:** MP3 (128kbps) یا WAV (44.1kHz)

---

## ۱. راهنماهای صوتی آواتار (Avatar Instructions)
این فایل‌ها در پوشه `res/raw` قرار می‌گیرند.

| نام فایل (ID) | متن دقیق جهت ضبط / ورودی TTS | کاربرد |
| :--- | :--- | :--- |
| `inst_type_word` | "قهرمان من، بیا این کلمه رو با هم بنویسیم!" | شروع هر مرحله آموزشی |
| `inst_find_char` | "می‌تونی حرفی که دنبالشیم رو روی دکمه‌ها پیدا کنی؟" | راهنمای صوتی هنگام مکث کاربر (Hint) |

## ۶. افکت‌های صوتی تعاملی (SFX)
این صداها به صورت غیرگفتاری (موزیکال یا افکت) هستند.

| نام فایل | نوع صدا / توصیف | کاربرد |
| :--- | :--- | :--- |
| `pop_sound` | صدای "پاپ" کوتاه و حبابی | هنگام لمس دکمه‌های کیبورد |
| `error_sound` | صدای لرزش ملایم یا "اوه-اوه" | هنگام تایپ اشتباه |
| `success_fest` | صدای تشویق و موزیک کوتاه پیروزی | پایان موفق یک کلمه یا عدد |
| `level_up` | صدای صعودی و جادویی (Magic) | زمان رشد گیاه یا تکامل جوجه |
| `island_unlock` | صدای باز شدن قفل (مثل باز شدن صندوقچه) | آزاد شدن جزیره جدید در نقشه |

## ۷. استراتژی قرائت متن (TTS Strategy) - کاملاً نیتیو و آفلاین
برای اینکه اپلیکیشن بتواند بدون اینترنت و به صورت نیتیو کلمات را به فارسی (گویش ایرانی) بخواند، موارد زیر الزامی است:

1. **نصب داده‌های صوتی (Voice Data):**
   - کاربر باید در تنظیمات گوشی به مسیر زیر برود:
     `Settings > Languages & Input > Text-to-speech output`
   - موتور `Google Services` را انتخاب کرده و در بخش `Install voice data` زبان `Persian (Iran)` را دانلود کند.
   
2. **الزامات پیاده‌سازی در کد:**
   - کلاس `TtsManager` همیشه وضعیت `isPersianReady` را بررسی می‌کند.
   - از متد `openTtsSettings()` برای راهنمایی کاربر جهت نصب دیتای آفلاین استفاده می‌شود.

3. **مزیت این روش:**
   - حفظ انعطاف‌پذیری کامل برای قرائت هر متن داینامیک.
   - عدم نیاز به ضبط هزاران فایل صوتی برای کلمات مختلف.
   - کاهش قابل توجه حجم نهایی فایل نصب (APK).

---
**نکته نهایی:** تمامی نام فایل‌ها باید با حروف کوچک انگلیسی باشند و هیچ فاصله یا کاراکتر خاصی نداشته باشند.
