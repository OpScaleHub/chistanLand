# لیست جامع دارایی‌های صوتی (Audio Assets) جهت ضبط یا TTS

این سند شامل تمامی فایل‌های صوتی مورد نیاز پروژه **چستان‌لند** است. برای بهترین خروجی، صداها باید با لحنی آرام، مهربان و تشویقی ضبط شوند. 

**تنظیمات پیشنهادی برای خروجی TTS:**
- **زبان:** فارسی (ایران)
- **سرعت:** ۰.۹ (کمی آهسته‌تر از نرمال)
- **فرمت:** MP3 (128kbps) یا WAV (44.1kHz)

---

## ۱. راهنماهای صوتی آواتار (Avatar Instructions)
این فایل‌ها در پوشه `res/raw` قرار می‌گیرند.

| نام فایل (ID) | متن دقیق جهت ضبط / ورودی TTS | کاربرد |
| :--- | :--- | :--- |
| `inst_type_word` | "قهرمان من، بیا این کلمه رو با هم بنویسیم!" | شروع هر مرحله آموزشی |
| `inst_find_char` | "می‌تونی حرفی که دنبالشیم رو روی دکمه‌ها پیدا کنی؟" | راهنمای صوتی هنگام مکث کاربر (Hint) |
| `inst_category` | "کدوم جزیره رو برای ماجراجویی امروز انتخاب می‌کنی؟" | صفحه انتخاب بین الفبا و اعداد |
| `inst_review` | "وقتشه بریم شهربازی و چیزهایی که یاد گرفتیم رو مرور کنیم!" | ورود به بخش مرور (Saga Map) |

## ۴. افکت‌های صوتی تعاملی (SFX)
این صداها به صورت غیرگفتاری (موزیکال یا افکت) هستند.

| نام فایل | نوع صدا / توصیف | کاربرد |
| :--- | :--- | :--- |
| `pop_sound` | صدای "پاپ" کوتاه و حبابی | هنگام لمس دکمه‌های کیبورد |
| `error_sound` | صدای لرزش ملایم یا "اوه-اوه" | هنگام تایپ اشتباه |
| `success_fest` | صدای تشویق و موزیک کوتاه پیروزی | پایان موفق یک کلمه یا عدد |
| `level_up` | صدای صعودی و جادویی (Magic) | زمان رشد گیاه یا تکامل جوجه |
| `island_unlock` | صدای باز شدن قفل (مثل باز شدن صندوقچه) | آزاد شدن جزیره جدید در نقشه |

## ۵. لیست کلمات جهت کلیک روی کارت (اختیاری)
اگر بخواهید با لمس کارت کلمه، تلفظ آن پخش شود:
*(نام فایل‌ها باید با پیشوند `word_` شروع شود، مثلاً `word_ab`)*

- کلمات: آب، بابا، باد، بام، سبد، نان، ابر، دست، بوم، سیب، باز، آش، کتاب، سگ، برف، شاخ، قایق، لباس، تاج، چای، کوه، ژله، صورت، ذرت، عینک، ثروت، حلزون، ضامن، طوطی، غذا، ظرف.
